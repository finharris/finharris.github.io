Higher or Lower Game

This is a simple number based higher or lower game. It asks the user to select the range to play in and allows the user to guess if the next randomly generated number will be higher or lower than the one currently displayed.

Search Engine Front-End

This is a front-end experiment to make a more functional search engine front end. It allows the user to type into the box without having the text box selected, press enter to search and clear the search box to allow the next search to happen sooner. It also has a couple useful tools.

three.js Test

A simple site mainly to test the use of three.js to render 3d objects realtime in the browser. The scene camera physically moves when the page is scrolled and the objects are animated to rotate.

Todo List App

This is a todo list app I created using react.js. It stores your todos in local storage so that if you refresh the page or delete the tab and come back later your todos will be saved. It also will swap to dark mode if the system setting prefers that.
